## Getting Started

1. Review the documentation in `/docs/design` to understand the system architecture
2. Install [KiCad](https://www.kicad.org/) to view and edit hardware designs
3. Check `/hardware/bom` for the list of required components
4. Refer to `/docs/assembly` for assembly instructions
